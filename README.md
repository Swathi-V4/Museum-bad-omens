# Cabinet of Omens

A digital museum website about bad-luck superstitions across cultures.

## Project concept
This project presents a small digital exhibit on how people use superstitions to manage fear, uncertainty, and risk. The site focuses on a few recognizable examples, including the number 13, black cats, cutting nails at night, and broken mirrors. Each exhibit card includes:

- a visual artifact area
- a historical explanation
- a psychological insight

## Required design framework

### Design style
Victorian fairy painting style.

This style shapes the site through:
- dark jewel-toned backgrounds
- glowing and ornate framed panels
- serif typography
- mystical and dreamlike atmosphere

### Cialdini principle
Authority.

The site uses a curator voice and structured museum labels so the visitor feels guided by an informed exhibition framework.

### Brand archetype
Magician.

The Magician archetype fits a museum about omens, hidden meaning, transformation, and mystery. It influences the tone, vocabulary, atmosphere, and emotional experience of the site.

## Orchestration process
This project follows a spec-driven workflow instead of asking AI for one final answer.

### Workflow
Spec → QA → Sprint → QA → Implement → QA

### How AI was used
1. Define the museum concept and audience.
2. Identify the exhibit structure for each superstition.
3. Lock the design direction: Victorian fairy painting.
4. Choose persuasion and archetype frameworks.
5. Build a first sprint version of the site.
6. Review the site as if it were a Smithsonian curator.
7. Revise for stronger narrative flow, visitor engagement, and exhibit coherence.

## Smithsonian-style review summary
The strongest parts of the site are:
- a clear central concept
- repeatable exhibit-card structure
- strong visual atmosphere
- a balance of folklore and psychology

Suggested next improvements:
- replace placeholder visuals with properly sourced images
- add captions and source credits
- add a bibliography or references page
- add one more superstition from a non-Western context for broader cultural range
